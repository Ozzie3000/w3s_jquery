JS and jQuery | .get() and .post() methods

    Some backround on the background
  • The .get() and .post() mthods are used for requesting data from a server using the HTTP GET or POST request
  • These two methods are known as request-reponse between the client-server
  • GET: Used for requesting data from a specific source
  • POST: Used to submit data to be processed at a specific resource
  • Think of get as retrieving information from some server
  • GET may return cached data
  • Think of post as giving information to some server
  • POST never caches data
    $.get()
  • The $.get() method is uses an HTTP protocol to make a GET request
  • syntax:
    $.get(URL, callback);
  • The URL paramter is required
  • The callback parameter is optional. It gets executed once the request is not processing
