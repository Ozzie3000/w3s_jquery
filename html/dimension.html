    Using dimension methods with jQuery
  • You have to remember in html there are 4 x 4 dimensions
    1. .width(); - set or gets the element only width
    2. .innerWidth(); - set or gets the element and padding width
    3. .outerWidth(); - set or gets the element, padding, and border width
    4. .outerWidth(true); - set or gets the element, padding, border, and margin width
    1. .height(); - set or gets the element only height
    2. .innerHeight(); - set or gets the element and padding height
    3. .outerHeight(); - set or gets the element, padding, and border height
    4. .outerHeight(true); - set or gets the element, padding, and border height
