    Fade
  • Fading elements into and out of visibility
  • 4 methods
    1. jQuery fadeIn()
    2. jQuery fadeOut()
    3. jQuery fadeToggle()
    4. jQuery fadeTo()
    fadeIn() method
  • Used to make hidden elements visible
  • Basic syntax:
  • $(selector).fadeIn(speed,callback); /// both time and callback are optional
    fadeOut() method
  • Used to make visible elments hidden
  • Basic syntax:
  • $(selector).fadeOut(speed,callback); /// both time and callback are optional
  • Fade out shapes above if visible
    fadeToggle() method
  • Used to toggle between fadeIn and fadeOut
  • Basic syntax:
  • $(selector).fadeToggle(speed,callback); /// both time and callback are optional
  • Toggle shapes above
