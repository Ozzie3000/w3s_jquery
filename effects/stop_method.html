JS and jQuery | stop method

    stop() method
  • This method is used to stop animations in their tracks
  • Obviously if it has finished you cannot stop it as it is no longer in progress
  • We will look at 2 examples
  • Stop a slide
  • Stop an animation with parameters
  • The stop method works on all jQuery effects such as custom animations, fading, or sliding
  • Basic syntax goes like this:
  • $(selector).stop(param1,param2);
  • param1 is optional. It specifies if the animation queue should be cleared.
    Default is false; so only current animation gets stopped
  • param2 is optional. It specifies if current animation should complete immediately or not
