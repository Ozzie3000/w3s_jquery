JS and jQuery | animate

    animate() method
  • Used for making custom animations
  • Basic syntax:
  • $(selector).animate({params},speed,callback);
  • params specifies CSS properties to be animated and is required
  • Speed is duration of time to complete effect and is optional
  • Callback is another function to be executed after completion and is also optional
  • Lets an take an HTML element that is static and manipulate it
  • You will need to first specify CSS position when creating it: absolute, fixed, relative
