jQuery selectors

    jQuery selectors
  • This is what allows your to find what you are looking for
  • All of them start like this $()
  • Most are based on CSS selectors
  • There are some custom ones as well
    Element selectors
  • Selecting things based on their element name
  • $("h1"), $("h2"), $("span")
    ID selectors
  • Selecting things based on their element id
  • Using the #
  • $("#id_1"), $("#id_2"), $("#id_3")
    Class selectors
  • Selecting things based on their element class
  • Using the .
  • $(".className_1"), $(".className_2"), $(".className_3")
