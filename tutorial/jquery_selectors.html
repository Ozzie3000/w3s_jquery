jQuery selectors

    jQuery selectors
  • This is what allows your to find what you are looking for
  • All of them start like this $()
  • Most are based on CSS selectors
  • There are some custom ones as well
    Element selectors
  • Selecting things based on their element name
  • $("h1"), $("h2"), $("span")
    ID selectors
  • Selecting things based on their element id
  • Using the #
  • $("#id_1"), $("#id_2"), $("#id_3")
    Class selectors
  • Selecting things based on their element class
  • Using the .
  • $(".className_1"), $(".className_2"), $(".className_3")
    More selectors
  • Besides the standard selectors you can get wild with your selections
  • A few below
    1. $("*") -- selects all elements
    2. $(this) -- used for selecting current element
    3. $("elmenent.className") -- selects a specified element such as h1 with class of className
    4. $("element:first") -- selects the first occurance of specified element
    5. $(" element_a element_b:first")
      $("h3 p:first") -- selects first paragraph from first h3
    6. $("ul li:first-child") -- Selects the first <li> element of every <ul>
    7. $("[href]") -- selects all elements with href
    8. $("a[target='_blank']") -- selects all anchors with the target attribute of open in new window
    9. $("a[target!='_blank']") -- selects all anchors not equal to open in new window
    10. $(":button") -- selects all buttons and input type of button
    11. $("li:even") -- selects all even list items
    12. $("li:odd") -- selects all odd list items
